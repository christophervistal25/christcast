# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

Be respectful. No harassment, discrimination, trolling, or personal attacks. Disagree on technical merits without making it personal.

## Scope

Applies to all project spaces: issues, pull requests, discussions, commit messages, and any communication where you represent the project.

## Enforcement

Report violations by opening a private security advisory on GitHub or emailing the maintainer. Reports are confidential. Maintainers may warn, remove content, or ban repeat offenders.

## Full text

The full Contributor Covenant v2.1 text is available at the link above. By participating in this project you agree to abide by it.
